# otdistfunc module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otdistfunc\examples\otdistfunc

Installation
  Silent install can be done with the switch /S (e.g. openturns-otdistfunc-0.0-py27.exe /S).
  To test otdistfunc module installation, in an OpenTURNS python console type:
    import otdistfunc
    print otdistfunc.__version__

Uninstallation
  OTDistFunc module will be automatically uninstalled at OpenTURNS removal.
  OTDistFunc module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
